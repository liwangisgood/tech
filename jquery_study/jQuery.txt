1、DOMd对象 原生js获取的对象就是DOM对象
	var myDiv=document.querySelector('div');

2、jQuery对象 用jQuery方法获取过来的对象就是jQuery对象。本质：通过$把DOM元素进行了包装
	$('div') //jQuery对象
3、jQuery对象只能使用jQuery方法，DOM对象则使用原生的JavaScript属性和方法
	myDiv.style.display='none' 如：不能$('div').style.display='none'、 也不能DOM对象myDiv.hide()用jQuery方法

// 需要有显示 有隐藏、筛选思想、排他思想、
    <script>
        $(function () {
            // 1.鼠标经过(mouseover)id是left的盒子里面的li
            $('#left li').mouseover(function () {
                // 2.得到当前对应li的索引号   $(this).index()
                var index = $(this).index();
                // 3.显示( show() )id是content的盒子的里面对应索引号的div
                $('#content div').eq(index).show();
                // 4.隐藏( hide() )其他id是content的盒子的里面的div
                $('#content div').eq(index).siblings('div').hide();
            });
        });
    </script>
